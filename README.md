# 小红书视频倍速扩展

为小红书网页版视频播放器添加额外的倍速选项。

## 功能

- 为小红书网页版视频播放器添加以下倍速选项：
  - 0.1x
  - 0.25x
  - 2.5x
  - 3x
  - 5x
  - 10x
  - 16x

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [Greasy Fork 安装页面](https://greasyfork.org/en/scripts/574801-%E5%B0%8F%E7%BA%A2%E4%B9%A6%E8%A7%86%E9%A2%91%E5%80%8D%E9%80%9F%E6%89%A9%E5%B1%95) 中的安装按钮
3. Tampermonkey 会自动弹出确认安装提示，点击确认即可

## 使用

1. 访问 [小红书网页版](https://www.xiaohongshu.com/)
2. 打开任意视频笔记
3. 点击视频播放器右下角的倍速按钮
4. 选择你想要的倍速

## 兼容性

- 支持所有现代浏览器（Chrome、Firefox、Edge、Safari）
- 需要安装 Tampermonkey 或类似的用户脚本管理器

## 注意事项

- 该脚本仅适用于小红书网页版
- 移动端应用不适用
